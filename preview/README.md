# Preview веб-интерфейса

Standalone HTML-копия страницы из `include/web_page.h`, в которой все
обращения к ESP32 заменены на локальные моки в JavaScript. Работает в
любом браузере **без** реального устройства, без Wokwi и без интернета.

## Как открыть

Просто **двойной клик** на `index.html` — откроется в браузере по умолчанию.

Никакой web-сервер не нужен — всё запускается через `file://`.

## Что работает

- ✅ Аналоговый циферблат (стрелки крутятся)
- ✅ Цифровое время и аптайм
- ✅ Установка времени через форму
- ✅ Будильник со срабатыванием (мигающее уведомление)
- ✅ Таймер обратного отсчёта со срабатыванием
- ✅ Журнал событий (последние 10)
- ✅ Кнопка сброса
- ✅ Тёмная/светлая тема (с сохранением в localStorage)
- ✅ Адаптивная вёрстка

## Что НЕ работает (по очевидным причинам)

- ❌ Звук зуммера (нет железа)
- ❌ Мигание LED (нет железа)
- ❌ Аппаратная кнопка (нет железа)
- ❌ NVS-сохранение (preview перезапускается с каждым обновлением страницы)
- ❌ HTTP API endpoints (`/api/time`, `/api/log`) — здесь нет HTTP-сервера

## Зачем это нужно

1. **Скриншоты для отчёта/презентации** — открыл, настроил тему, сделал
   красивые скрины.
2. **Демонстрация на защите**, если нет железа и Wokwi не открывается
   (плохой интернет, заняты серверы).
3. **Проверить как выглядит интерфейс на разных устройствах** — открой
   с телефона по локальной сети или просто сожми окно браузера.

## Чем отличается код от настоящей версии

В `index.html` находится почти полная копия HTML/CSS из `web_page.h`,
но JavaScript изменён:

| Настоящая версия | Preview |
|------------------|---------|
| `await fetch('/api/time')` | локальная функция `getCurrentTime()` |
| `await fetch('/api/log')` | локальный массив `logBuf` |
| `await fetch('/set?time=...')` | прямое изменение `baseSeconds` и `syncMillis` |
| Будильник срабатывает на ESP32, JS только показывает уведомление | Будильник полностью обрабатывается в JS |

Логика расчёта времени **одинаковая** в обоих версиях — это копия
формулы из `main.cpp`:

```
currentTime = baseSeconds + (now - syncMillis) / 1000
```

Это удобно при объяснении на защите — можно открыть preview и main.cpp
рядом и показать, что JS-версия в точности повторяет С-версию.
